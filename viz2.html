About the Pearson's correlation coefficient

  • No correlation: If r is exactly 0.
  • Weak positive correlation: If r is greater than 0 but less than 0.3.
  • Moderate positive correlation: If r is between 0.3 (inclusive) and 0.7.
  • Strong positive correlation: If r is 0.7 or greater.
  • Weak negative correlation: If r is less than 0 but greater than -0.3.
  • Moderate negative correlation: If r is between -0.3 (inclusive) and -0.7.
  • Strong negative correlation: If r is -0.7 or less.
